The ddnet 0.6 network protocol

The teeworlds modification DDraceNetwork is based on the teeworlds 0.6 protocol. While being fully teeworlds compatible ddnet added its own extensions. Meaning vanilla 0.6 teeworlds clients can join ddnet servers and ddnet clients can join 0.6 vanilla teeworlds servers. But when a ddnet client connects to a ddnet server. They start talking the ddnet protocol which for example adds security tokens, new game and system messages and new snap items. This section only covers the ddnet extensions so for a complete protocol overview you also have to look at the teeworlds 0.6 protocol.
